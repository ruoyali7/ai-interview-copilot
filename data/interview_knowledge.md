# AI Interview Knowledge Base

## RAG Questions
Ask candidates to explain embeddings, vector search, chunking, retrieval, reranking, and hallucination reduction.

## Backend Questions
Ask candidates about REST APIs, scalability, caching, async processing, database design, and microservices.

## System Design Questions
Ask candidates to design a document Q&A system, AI interview assistant, or scalable LLM application.

## Evaluation Questions
Ask candidates how to evaluate LLM outputs using relevance, faithfulness, correctness, and LLM-as-a-judge.

## Frontend System Design
Ask candidates how they would optimize React rendering, frontend state management, browser performance, and UI responsiveness.

## Backend System Design
Ask candidates how they would design scalable APIs, microservices, caching, async queues, and database schemas.

## AI System Design
Ask candidates how they would design RAG pipelines, embedding retrieval, vector databases, reranking, hallucination reduction, and LLM evaluation.

## Generic System Design
Ask candidates how they would design a general software system, explain tradeoffs, handle scale, and monitor performance.

## Cloud Infrastructure
Ask candidates how they would deploy services using AWS, Kubernetes, CI/CD pipelines, load balancers, and observability.

## Resume Writing
Ask candidates how they would improve resume bullet points, quantify impact, and tailor experience to a job description.

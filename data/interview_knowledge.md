# AI Interview Knowledge Base

## RAG Questions
Ask candidates to explain embeddings, vector search, chunking, retrieval, reranking, and hallucination reduction.

## Backend Questions
Ask candidates about REST APIs, scalability, caching, async processing, database design, and microservices.

## System Design Questions
Ask candidates to design a document Q&A system, AI interview assistant, or scalable LLM application.

## Evaluation Questions
Ask candidates how to evaluate LLM outputs using relevance, faithfulness, correctness, and LLM-as-a-judge.
